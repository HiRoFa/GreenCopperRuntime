Struct MySqlTime

pub struct MySqlTime { /* private fields */ }
Expand description

Container for a MySQL TIME value, which may be an interval or a time-of-day.

Allowed range is -838:59:59.0 to 838:59:59.0.

If this value is used for a time-of-day, the range should be 00:00:00.0 to 23:59:59.999999. You can use Self::is_valid_time_of_day() to check this easily.

  • MySQL Manual 13.2.3: The TIME Type
  • MariaDB Manual: TIME

ยงErrors
  • MySqlTimeError::NegativeZero if all fields are 0 but sign is MySqlTimeSign::Negative.
  • MySqlTimeError::FieldRange if any field is out of range:
    • hours > 838
    • minutes > 59
    • seconds > 59
    • microseconds > 999_999
  • MySqlTimeError::SubsecondExcess if the magnitude is less than one second over the maximum.
    • Durations 839 hours or greater are covered by FieldRange.

ยงNote: Precision Truncation

See also: MySQL Manual, section 13.2.6: Fractional Seconds in Time Values
