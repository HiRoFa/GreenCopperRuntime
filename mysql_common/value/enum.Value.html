Enum mysql_common::value::Value

source · 
pub enum Value {
    NULL,
    Bytes(Vec<u8>),
    Int(i64),
    UInt(u64),
    Float(f32),
    Double(f64),
    Date(u16, u8, u8, u8, u8, u8, u32),
    Time(bool, u32, u8, u8, u8, u32),
}
Expand description

Client side representation of a value of MySql column.

The Value is also used as a parameter to a prepared statement.
